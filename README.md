# Mihaitzuuu Portfolio — Starter Scaffold

This repo is a starter scaffold for the cinematic Minecraft portfolio for `Mihaitzuuu`.

Quick start:

```bash
npm install
npm run dev
```

What I included:

- Next.js TypeScript scaffold
- TailwindCSS config with design tokens
- Core components: `Navbar`, `Hero`, `AboutSection`, `BuildCategories`, `GallerySection`, `ServerLifeSection`, `ContactSection`, `Footer`
- Placeholder hero SVG
- Legal pages: `404`, `Terms of Service`, `Cookie Policy`

Next steps I can take now:
